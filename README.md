# Linear Equation Solver using MATLAB
## Description
A MATLAB application for solving systems of linear equations.
## Features
- Solves 2×2 and 3×3 linear systems
- Accepts matrix coefficients as input
- Displays solution vector
- GUI developed using MATLAB App Designer
## Technologies Used
- MATLAB
- App Designer
## Example
Ax = b
A = [2 3; 1 4]
b = [5; 6]
Solution:
x = [0.4; 1.4]
